📋 Run Summary
Forest_Night_01 completed a 24-hour simulation using BirdNET v2.4 on Radxa Zero hardware with the Tropical Night scenario pack. The system achieved 94.2% detection accuracy across 127 events, slightly below the expected 130 — 3 low-confidence bird calls were missed during heavy rain periods.

The primary concern is a network bottleneck at relay nodes R1 and R2, which handle 78% of total traffic. R1's battery dropped to 23% due to 3.2× higher drain from continuous relaying. If R1 fails, 4 downstream sensors would go offline.

Recommendation: Add a relay node between R1 and the south sensor cluster to reduce R1 congestion by ~40% and improve fault tolerance. The AI false positive rate of 6.8% is slightly above the 5% target — most false positives occurred during high-wind periods misclassified as chainsaw activity.
